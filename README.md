# vikop5portfolio

Welcome to a digital experience heavily inspired by the bold, high-contrast aesthetic of Persona 5, where sleek typography and jagged UI elements guide you through my professional Metaverse. 

## Tech Stacks :
 UI Framework & Styling: Tailwind CSS & Nes.css
 Front-end: HTMX, Vanilla JS (ES6+)
 Animation Engines: GSAP and Anime.js
 Audio: Web Audio API
 

This portfolio serves as a stylistic "Calling Card" featuring a deep dive into my About section, a breakdown of my technical Skills, and a chronological log of my Education and Experience, alongside a Projects compendium showcasing my latest work—all rounded out with a Contact portal for future collaborations.
